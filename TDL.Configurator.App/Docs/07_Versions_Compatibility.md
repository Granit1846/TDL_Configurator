# Версии, совместимость и типовые проверки

Эта страница — короткий чеклист, чтобы не ловить «магические» ошибки из‑за несовместимости версий.

---

## 1) Skyrim SE vs AE
TDL‑плагин под SKSE должен соответствовать вашей версии игры (runtime).
Если DLL собрана под другой runtime — возможны:
- плагин не загрузится,
- игра упадёт при старте,
- команды не будут приниматься.

---

## 2) Где проверить загрузку SKSE‑плагина
### SKSE лог
Обычно лежит в:
- `Documents\My Games\Skyrim Special Edition\SKSE\`

Ищите строки вида:
- «... loaded correctly ...» для `TDL_StreamPlugin.dll`

### Лог самого плагина
Если включён файловый лог, он обычно лежит там же:
- `...\SKSE\TDL_StreamPlugin.log`

---

## 3) Если используете Vortex/MO2
Чеклист:
- совпадает ли папка запуска Skyrim с тем, куда вы сохраняете INI,
- реально ли `tdl_send.exe` доступен в активной Data.
